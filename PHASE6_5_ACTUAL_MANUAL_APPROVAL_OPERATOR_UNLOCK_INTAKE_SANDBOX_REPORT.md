# Phase 6.5 — Actual Manual Approval / Operator Unlock Intake Sandbox

This phase adds a review-only sandbox that defines how real manual approval and operator unlock files would be detected and reviewed before a later Phase 7 signed testnet validation design.

It does not create `storage/manual_approval/approval_intake_submission.json`.
It does not create `storage/latest/operator_unlock_request.json`.
It does not enable signed testnet execution.
It does not allow testnet order submission.

Expected default status: `PHASE6_5_ACTUAL_APPROVAL_OPERATOR_UNLOCK_INTAKE_SANDBOX_BLOCKED_REVIEW_ONLY`.
